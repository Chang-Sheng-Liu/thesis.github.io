This flow chart shows how the FSM works.

There are the following steps the OCG module should contain :

  • start OCG
  • get option
  • detect file
  • initiate an emulation
  • parse filename
  • create directory
  • parse XML
  • save XML
  • call OAI emulator
  • generate report
  • end OCG
