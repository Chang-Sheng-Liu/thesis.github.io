◆ set_permissions_map()

int set_permissions_map ( void  )

The permissions map is a bit map which defines which modes are available to a user. The permissions groups map to linux groups in the following way: subtract 1000. The remainder is the bit number. Thus unix groups 1000, 1001, 1002 etc represent the permissions groups default, config and admin respectively.
