Collaboration diagram for Trace-based Mobility Model:

The trace-based mobility model allows a user to provide external mobility traces, eithe gathered from realistic monitoring, from an different simulator or specififed according to particular scenarios. The trace format follows that of <TIME> <ID> <X> <Y> <SPEED>, but new parser could easily be added.
Node that this requires the user to specify in OCG the name of the trace file, which should be located user the ./TRACE folder.
